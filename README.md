## 🔗 Architecture Multi-Machines (du smartphone au cluster)

**Du plus petit appareil au cluster complet — tout se connecte ensemble.**

- 🖥️ **Cluster de machines maillées** (orchestration containerisée), rôles asymétriques : un nœud pilote, les autres calculent
- 📱 **Android & appareils peu puissants** : un smartphone ou un vieux PC devient un client — **aucun GPU requis**, tout le calcul est absorbé par le cluster
- 🔌 **Connexion de plusieurs machines ensemble** : liaisons sécurisées (tunnels SSH chiffrés), exécution parallèle
- ⚡ **VRAM dynamique** : modèles chargés/déchargés à la volée, zéro gaspillage matériel
